# 任务中心功能说明

## 页面能力

`/tasks` 是当前工作台的任务管理页，默认展示爆款素材分析记录。

当前支持：

- 任务列表
- 关键词搜索
- 类型筛选：`viral` / 爆款素材分析
- 清空筛选
- 加载更多
- 展开摘要详情
- 进入 `/tasks/[id]` 查看完整详情
- 删除单条记录

`/tasks/[id]` 详情页支持：

- 返回任务中心
- 查看记录 ID、创建时间、类型、平台、来源
- 查看输入素材和一句话摘要
- 查看结构化建议
- 查看格式化 JSON
- 删除记录后跳回 `/tasks`
- 记录不存在时显示友好提示

## API

### GET /api/tasks

示例：

```text
GET /api/tasks?type=viral&q=&limit=10&offset=0
```

参数：

- `type`：任务类型，默认 `viral`
- `q`：关键词，trim 后搜索
- `limit`：每次返回数量，默认 `10`，最小 `1`，最大 `50`
- `offset`：偏移量，默认 `0`，最小 `0`

返回结构：

```json
{
  "ok": true,
  "records": [],
  "data": {
    "items": []
  },
  "page": {
    "type": "viral",
    "q": "",
    "limit": 10,
    "offset": 0,
    "total": 0,
    "hasMore": false,
    "nextOffset": null
  }
}
```

说明：

- `records` 是新列表字段。
- `data.items` 保留兼容旧前端读取方式。
- 排序为 `createdAt desc`。
- 搜索优先覆盖标题、链接、输入素材、一句话判断、等级和结果 JSON 文本。

### GET /api/tasks/[id]

读取单条任务记录详情。

### DELETE /api/tasks/[id]

删除单条任务记录。

删除前端必须先 `window.confirm`，避免误删。

## 验收清单

本地或线上验收时建议使用唯一测试前缀：

```text
LOCAL_TEST_STEP_77_时间戳
```

必须验证：

- `/viral` mock 生成并保存
- `/tasks` 默认列表可用
- 搜索存在关键词有结果
- 搜索不存在关键词显示无结果
- 清空后恢复列表
- 加载更多可用
- 展开详情可用
- 查看详情可进入 `/tasks/[id]`
- 详情页返回可用
- 删除本轮测试记录后列表不再显示

## 安全注意事项

- 只用 mock 结果做任务闭环验收。
- 不点击 AI 深度拆解按钮。
- 不调用真实 AI。
- 不读取、打印或修改 `.env.local`。
- 不删除旧记录或真实业务记录。
- 删除测试数据时，只删除本轮新建且带唯一测试前缀的记录。
