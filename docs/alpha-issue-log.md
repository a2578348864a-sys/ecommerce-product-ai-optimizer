# Alpha 问题记录表

> **用途**：测试过程中发现的任何问题，请在此记录。
>
> **填写规则**：
> - 每个问题独立一行记录
> - 先判定问题等级，再写描述
> - 如果同一问题多次发生，合并到一条，备注发生次数

---

## 问题等级速查

| 等级 | 含义 | 举例 |
|------|------|------|
| **P0** | 核心流程中断 | 页面 500、保存失败、详情页打不开、"任务记录不存在或已删除"、listing-copy 出现虚假认证声明（如 ASTM certified / FDA approved 等没有证书却写出来的）、严重串货 |
| **P1** | 分析结论明显错误 | 高风险商品被推荐为"新手可小单测试"、明显风险被漏掉（如儿童磁铁玩具没提磁铁吞咽风险）、summary 结论与 risk 结论矛盾 |
| **P2** | 质量一般但不影响主流程 | 结论偏保守、文案不够好、推荐太笼统不够具体、展示不够清楚、个别字段判断偏轻或偏重 |
| **S1** | 服务稳定性问题 | AI API 502/500、请求超时、服务器资源耗尽、需要 PM2 restart 才能恢复 |
